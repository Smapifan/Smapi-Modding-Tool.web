# Tilesheets

Place your Stardew Valley tilesheet PNG images here.

The editor will automatically discover and load these images when a `.tbin` map
references them (matching by the tilesheet ID or filename).

## Where to get the images

Extract PNG files from your Stardew Valley installation:

- **Windows**: `%ProgramFiles(x86)%\Steam\steamapps\common\Stardew Valley\Content\Maps\`
- **macOS**: `~/Library/Application Support/Steam/steamapps/common/Stardew Valley/Content/Maps/`
- **Linux**: `~/.local/share/Steam/steamapps/common/Stardew Valley/Content/Maps/`

You can use [StardewXnbHack](https://github.com/Pathoschild/StardewXnbHack) to
unpack `.xnb` files into PNG if needed.

## Expected file names

| File name                      | Used by                              |
|-------------------------------|--------------------------------------|
| `spring_outdoors.png`         | Outdoor maps (spring season)         |
| `summer_outdoors.png`         | Outdoor maps (summer season)         |
| `fall_outdoors.png`           | Outdoor maps (fall season)           |
| `winter_outdoors.png`         | Outdoor maps (winter season)         |
| `mine_stone_tile_sheet.png`   | Mine levels                          |
| `Flooring.png`                | Interior floors                      |
| `walls_and_floors.png`        | Interior walls/floors                |
| `paths.png`                   | Path tiles                           |
| `furniture.png`               | Furniture                            |
